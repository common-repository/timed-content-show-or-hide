== Description ==
This plugin show or hide the content after a specified time.

Example:

[timed-content time="10000" visible="yes"]The content[/timed-content]

== Installation ==

1. Extract the plugin folder from the downloaded ZIP file.
2. Upload timed content show or hide to your /wp-content/plugins/ directory.
3. Activate the plugin from the "Plugins" page in your Dashboard.
